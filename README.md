# Travel page layout

## Description
Main page of travel site, made by using Figma template (original-picture.png).

## Technologies
- HTML5;
- CSS3 (Grid/Flexbox);
- JS.

## Additional
- adaptive layout with using "@media";
- Burger-menu for mobile;
- EN/RU site language script realization;
- using "tooltip".

## How to run
1. Copy project;
2. Open "index.html" in browser.
